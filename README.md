# github.com-GravityEscape-privacy-policy
개인정보처리방침

Gravity Escape는 사용자의 개인정보를 직접 수집하거나 저장하지 않습니다.

본 앱은 광고 제공을 위해 Google AdMob을 사용할 수 있습니다.

Google AdMob은 광고 제공 및 성능 분석을 위해 기기 정보, 광고 식별자 등의 정보를 수집할 수 있습니다.

수집되는 정보는 Google의 개인정보처리방침에 따라 처리됩니다.

Google 개인정보처리방침:
https://policies.google.com/privacy

본 앱은 다음 정보를 수집하지 않습니다.

- 이름
- 이메일
- 전화번호
- 위치정보
- 연락처

문의:
이메일 주소 기재
